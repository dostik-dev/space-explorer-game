# Космический исследователь (Space Explorer)

Игра на Python с графической библиотекой Arcade. Игрок управляет кораблём, исследует галактику, добывает артефакты и улучшает характеристики.

##  О проекте
Жанр: космическая стратегия / RPG.  
Цель: исследовать 16 звёздных систем, собирать артефакты, зарабатывать кредиты и улучшать корабль.  
Прогресс сохраняется в базе данных SQLite.

##  Основные фичи
- 4 расы с уникальными бонусами
- Карта из 16 систем с ветвящимися связями
- Улучшения корабля (энергия, щиты, сканер, скорость)
- Добыча артефактов (COMMON, RARE, EPIC, LEGENDARY)
- Случайные события и экономическая система
- Автосохранение в SQLite

## Стек
- Python 3
- Arcade
- SQLite
- PyQt (использовался в прототипе)

## Архитектура
Проект разбит на 5 модулей:
- `main.py` — точка входа
- `game_core.py` — ядро с логикой
- `ui.py` — графический интерфейс
- `database.py` — работа с БД
- `models.py` — классы Artifact, StarSystem

## Запуск
pip install arcade
python main.py

Требуется Python 3.9+
